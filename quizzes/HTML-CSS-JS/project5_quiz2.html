1. What will be the output of the following code?

    sum += i;
}

2. What is the output of the following while loop?

3. What will the output of the following code be?

for (i = 0; i < 3; i++) {
    console.log(i);
    while (i == 1) {
        console.log("Inside while loop");
        break;
    }
}
