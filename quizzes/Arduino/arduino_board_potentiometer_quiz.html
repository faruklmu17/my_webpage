1. What is a potentiometer primarily used for in a circuit?

A potentiometer is a three-terminal resistor with a sliding or rotating contact that forms an adjustable voltage divider.

2. How many terminals (pins) does a standard potentiometer have?

A standard potentiometer has three pins: two outer pins for the full resistance and a middle pin (wiper) for the variable output.

3. Which Arduino function is used to read the value of a potentiometer?

The analogRead() function reads the voltage applied to an analog input pin.

4. What is the range of integer values returned by analogRead() on an Arduino Uno?

Arduino Uno has a 10-bit ADC, which yields 2^10 = 1024 unique values (0 to 1023).

5. Which terminal of the potentiometer is usually connected to the Arduino's analog input pin?

The middle pin (wiper) provides the variable voltage that the Arduino reads.

6. If a potentiometer's outer pins are connected to 5V and GND, what voltage range is available at the wiper?

As you turn the knob, the wiper moves between the 5V and GND terminals, creating a variable voltage.

7. What does the map() function do in typical potentiometer projects?

map(value, fromLow, fromHigh, toLow, toHigh) is used to re-proportion the input range to an output range.

8. What component inside the Arduino chip converts the analog voltage to a digital number?

The ADC is the specialized hardware that measures voltage and turns it into a number.

9. To control the brightness of an LED (0-255) using a potentiometer (0-1023), what formula or function is common?

Dividing 1023 by 4 gives approximately 255, scaling the input to the PWM range.

10. What electrical configuration is a potentiometer using when its wiper provides a variable voltage?

A potentiometer acts as two resistors in series, with the wiper tapping the voltage between them.

11. If you connect a 10k ohm potentiometer across 5V and GND, how much total resistance is between the 5V and GND pins?

The resistance between the two outer pins is fixed at the rated value of the potentiometer.

12. What happens to the analogRead() value if you leave an analog pin "floating" (connected to nothing)?

Floating pins pick up EMI and noise, causing unpredictable readings.

13. Potentiometers are commonly used for which of the following?

Potentiometers are versatile tools for any application requiring user-adjustable analog input.

14. In Arduino code, analog pins (A0-A5) need to be initialized with pinMode(pin, INPUT). Is this true?

While you can use pinMode, it's not strictly necessary for analogRead(), and pins A0-A5 can also function as digital pins.

15. What is the maximum voltage an Arduino Uno analog pin can safely measure by default?

By default, the reference voltage (Vref) is 5V on an Uno. Exceeding this can damage the pin.
