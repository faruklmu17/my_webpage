Privacy Policy

This Privacy Policy applies to the Playwright Test Results Badge Chrome extension.

Data Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data. All functionality runs locally within the user’s browser.

The extension reads a user-provided Playwright JSON test report (from a public GitHub URL or a local file) solely to display test result information in the Chrome toolbar badge and popup.

Data Usage

The extension does not send any data to external servers, does not use analytics or tracking technologies, and does not communicate with third-party services.

Information Not Collected

The extension does not collect or process any of the following:

  • Personally identifiable information (such as name, email, or address)
  • Authentication information (passwords, tokens, or credentials)
  • Browsing history or website activity
  • Location data
  • Financial or payment information
  • Health-related information
  • Personal communications
  • User activity data (clicks, keystrokes, or tracking)

Permissions

The permissions requested by the extension are used only to fulfill its single purpose of displaying Playwright test results. No permissions are used to collect or transmit user data.

Changes to This Policy

If the extension’s behavior changes in the future, this privacy policy will be updated accordingly.

Contact

If you have any questions about this privacy policy, please contact the developer via the Chrome Web Store listing.
