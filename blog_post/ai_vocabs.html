Essential & Emerging AI Vocabulary Glossary

📚 Comprehensive Guide
🤖 AI & ML Terms

A comprehensive guide to the most important AI and ML terms — from basics to cutting-edge concepts like context engineering and LLMs.

1. Core AI & Machine Learning Concepts

Artificial Intelligence (AI)

The simulation of human intelligence in machines that are programmed to think and learn like humans.

Machine Learning (ML)

A subset of AI that enables computers to learn and improve from experience without being explicitly programmed.

Supervised Learning

Learning with labeled training data where the algorithm learns to map inputs to correct outputs.

Unsupervised Learning

Learning from data without labeled examples, finding hidden patterns or structures in data.

Reinforcement Learning

Learning through interaction with an environment, receiving rewards or penalties for actions taken.

Deep Learning

A subset of ML using neural networks with multiple layers to model complex patterns in data.

Neural Network

A computing system inspired by biological neural networks, consisting of interconnected nodes (neurons).

Overfitting

When a model learns training data too well, performing poorly on new, unseen data.

Underfitting

When a model is too simple to capture underlying patterns, performing poorly on both training and test data.

Regularization

Techniques used to prevent overfitting by adding constraints or penalties to the model.

2. Advanced Learning Techniques

Transfer Learning

Using knowledge gained from one task to improve performance on a related but different task.

Few-shot Learning

Learning to perform tasks with only a few examples, mimicking human ability to generalize quickly.

Zero-shot Learning

Performing tasks without any specific training examples, using only general knowledge.

Self-supervised Learning

Learning representations from unlabeled data by creating supervision signals from the data itself.

Contrastive Learning

Learning by comparing similar and dissimilar examples to create meaningful representations.

Multi-modal Learning

Learning from multiple types of data (text, images, audio) simultaneously to improve understanding.

3. Large Language Models (LLMs)

LLM (Large Language Model)

AI models trained on vast amounts of text data to understand and generate human-like language.

Prompt Engineering

The art of crafting effective input prompts to get desired outputs from language models.

Few-shot Prompting

Providing a few examples in the prompt to guide the model's response format and style.

Zero-shot Prompting

Asking the model to perform a task without providing any examples in the prompt.

Chain-of-Thought Prompting

Encouraging the model to show its reasoning process step-by-step before giving an answer.

Function Calling

The ability of LLMs to invoke external tools or APIs to perform specific tasks.

System Prompt

Initial instructions that define the AI's role, behavior, and constraints for a conversation.

Context Window

The maximum amount of text (tokens) that a language model can process at once.

Token

The basic unit of text processing in LLMs, typically representing words or parts of words.

Context Engineering

The strategic design and optimization of context provided to AI models to improve their performance and accuracy.

4. Retrieval & Embeddings

RAG (Retrieval-Augmented Generation)

Combining information retrieval with text generation to provide more accurate and up-to-date responses.

Embeddings

Numerical representations of text, images, or other data that capture semantic meaning in vector space.

Vector Database

Specialized databases designed to store and efficiently search through high-dimensional vector embeddings.

Semantic Search

Search that understands the meaning and context of queries, not just keyword matching.

Chunking

Breaking down large documents into smaller, manageable pieces for processing and retrieval.

5. Generative AI

Generative AI

AI systems that can create new content like text, images, music, or code based on training data.

Diffusion Models

Generative models that create images by gradually removing noise from random data.

GAN (Generative Adversarial Network)

Two neural networks competing against each other to generate increasingly realistic synthetic data.

Latent Space

A compressed representation of data where similar items are located close to each other.

Text-to-Image

AI systems that generate images from textual descriptions, like DALL-E or Midjourney.

Text-to-Video

AI models that create video content from text prompts, representing the next frontier in generative AI.

Style Transfer

Applying the artistic style of one image to the content of another image using AI.

Prompt Chaining

Using the output of one AI prompt as input for another to create complex, multi-step workflows.

6. AI Agents and Automation

AI Agent

An autonomous system that can perceive its environment, make decisions, and take actions to achieve specific goals.

Planning / Reasoning Loop

The cyclical process where AI agents analyze situations, plan actions, execute them, and learn from results.

Toolformer

AI models that can learn to use external tools and APIs to enhance their capabilities beyond text generation.

AutoGPT

An autonomous AI agent that can break down complex tasks into subtasks and execute them independently.

ReAct (Reason + Act)

A framework combining reasoning and acting, allowing AI to think through problems and take appropriate actions.

7. Ethics and Safety

Explainable AI (XAI)

AI systems designed to provide clear explanations for their decisions and reasoning processes.

Model Interpretability

The ability to understand and explain how AI models make decisions in human-understandable terms.

Bias in AI

Unfair or discriminatory outcomes in AI systems due to biased training data or algorithmic design.

Fairness

Ensuring AI systems treat all individuals and groups equitably without discrimination.

Hallucination

When AI models generate false or nonsensical information that appears plausible but is factually incorrect.

Alignment

Ensuring AI systems behave in accordance with human values and intended objectives.

Red Teaming

Adversarial testing of AI systems to identify vulnerabilities, biases, and potential harmful behaviors.

Constitutional AI

Training AI systems with a set of principles or "constitution" to guide ethical behavior and decision-making.

Watermarking

Embedding invisible markers in AI-generated content to identify it as artificially created.

8. MLOps and Deployment

Model Inference

The process of using a trained AI model to make predictions or generate outputs on new data.

Model Serving

Deploying trained models in production environments to handle real-time requests and predictions.

Latency

The time delay between input submission and output generation in AI systems, crucial for user experience.

Model Drift

The degradation of model performance over time due to changes in data patterns or distributions.

Data Leakage

When information from outside the training dataset inadvertently influences the model, leading to overly optimistic results.

A/B Testing

Comparing two versions of an AI model or system to determine which performs better in real-world conditions.

AutoML

Automated machine learning that streamlines the process of applying ML to real-world problems with minimal human intervention.

Quantization

Reducing the precision of model weights and activations to decrease model size and improve inference speed.

Edge AI

Running AI computations locally on devices rather than in the cloud, enabling faster responses and better privacy.
